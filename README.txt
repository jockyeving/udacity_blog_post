The dataframe used can be accessed publicly on https://www.kaggle.com/datasets/stackoverflow/so-survey-2017?resource=download

The following notebooks are used to create visuals in answering the following questions:
1. What programming languages are used most among responders, and what trends are to be expected?
2. How does a responders social status, or habits influence his/her job satisfaction?
3. How does the idea of being underpaid/overpaid affect an individual's immersion in his/her job?
